### git中的main分支      
```
由于某些原因,git上的master分支默认改为了main分支,所以我们在push的时候回出现问题      
git branch  查看当前分支     
git branch -m master main 把master重命名为main分支        
git pull origin main --allow-unrelated-histories 【允许不相关历史提交，并强制合并】          
git add .       
git  commit  -m  " xxx "          
git push origin main   
```
