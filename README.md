### git中的main分支      

由于某些原因,git上的master分支默认改为了main分支,所以我们在push的时候回出现问题      

git branch  查看当前分支     
git branch -m master main 把master分支切换为main分支        
git pull origin main --allow-unrelated-histories  拉取代码    
git add .       
git  commit  -m  " xxx "          
git push origin main
